# Fibonacci

A C# console application that prints out a fibonacci sequence and fibonacci tree.

## Getting Started

### Prerequisite
Install .NET Core\
[Download here](https://dotnet.microsoft.com/download)

### To run the program
1. Clone this respository
2. Open command terminal
3. Change directory into **FibonacciTree** folder
4. run command\
 ```dotnet run```

### To use the program
Enter a negative number to quit.\
Enter a positive number to display the fibonacci tree.\
**Start with a very small number first**
